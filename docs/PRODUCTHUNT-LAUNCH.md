# ProductHunt Launch — Built By AI

## Launch Post

**Tagline:** We build your web app while you wait. $100-$500, fixed price.

**Story:**

Most people who want to "build something with AI" spend more time setting up Cursor, debugging prompts, and iterating than they would have spent just hiring a developer.

The problem isn't access to AI coding tools. It's the time and frustration of using them.

Built By AI is a small studio that uses Cursor, Claude Code, and Codex to build working software for people who have ideas but don't want to become developers.

**How it works:**
1. You describe what you want (plain English is fine)
2. We confirm scope and send a Stripe payment link (no commitment until you pay)
3. We build it using AI coding tools and deliver a GitHub repo with working code
4. You own everything — forever

**Pricing:** $100 / $250 / $500 fixed tiers

**What makes it different:**
- No retainers, no hourly billing
- You get the code, not a SaaS subscription
- Fixed revisions window, then it's yours
