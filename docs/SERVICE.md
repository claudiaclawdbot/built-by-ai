# Built By AI — Service Description

## The Offer

Built By AI is a boutique AI-assisted development service. We take your project idea and deliver working, deployed code — faster and cheaper than traditional agencies or freelancers.

**Core promise:** Tell us what you want built. We use AI coding tools to build it. You get working software.

---

## How It Works

1. **Submit** — Fill out the intake form at builtbyai.dev with your project description
2. **Scope & Pay** — We confirm it fits a tier, you pay via Stripe
3. **Build** — We code it and push to a private GitHub repo
4. **Review** — You test it, request changes within the revision window
5. **Done** — Code is yours forever

---

## Pricing Tiers

| Tier | Price | Scope | Delivery |
|------|-------|-------|----------|
| **Basic** | $100 | Single page or simple feature, frontend only (~200 LOC) | 3–5 days |
| **Standard** | $250 | Up to 3 pages or complex feature, frontend + backend (~600 LOC) | 5–7 days |
| **Complex** | $500 | Up to 6 pages or full feature module, full-stack + DB (~1500 LOC) | 10–14 days |
| **Custom** | Variable | Larger projects scoped individually | TBD |

### What's Included in Every Tier
- Private GitHub repository (you own it)
- Clean, readable code
- Revision window per tier
- Direct communication with the builder

### What's NOT Included
- Monthly hosting fees (deploy to Vercel/Netlify free tier — ~$0)
- Ongoing maintenance after delivery
- Scope creep beyond tier limits (quoted separately)

## Technologies

We work with:
- **Frontend:** Next.js, React, Tailwind CSS, HTML/CSS/JS
- **Backend:** Node.js, API routes
- **Database:** Supabase, Firebase (free tiers)
- **Deployment:** Vercel, Netlify (free tiers)
- **AI Tools:** Claude Code, Cursor, GPT-5, Ollama
